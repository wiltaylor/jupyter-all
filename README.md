# Jupyter All
This repository has a Juypter Lab environment with the following Kernels installed: 
- [Bash](https://github.com/takluyver/bash_kernel)
- [C++](https://github.com/jupyter-xeus/xeus-cling)
- [C#, PowerShell and F#](https://github.com/dotnet/interactive/)
- [Go](https://github.com/janpfeifer/gonb)
- [Graphviz dot](https://github.com/laixintao/jupyter-dot-kernel)
- [GNUPlot](https://github.com/has2k1/gnuplot_kernel)
- [Lua](https://github.com/guysv/ilua)
- [IRuby](https://github.com/SciRuby/iruby)
- [Rust](https://github.com/SciRuby/iruby)
- [Typescript/Javascript](https://github.com/yunabe/tslab)
